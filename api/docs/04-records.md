# Managing Transactions (Records)

## What is a Record?

A record (or transaction) is any money movement: an expense, income, or transfer.

## Transaction Types

- **Expense**: Money going out (purchase, payment, bill).
- **Income**: Money coming in (salary, refund, gift).
- **Transfer**: Movement between your own accounts.

## Creating a Record Manually

1. Go to the **Records** section in the sidebar.
2. Click the **+** button or **Add Record**.
3. Fill in the fields:
   - **Date**: Transaction date.
   - **Name**: Description (e.g., "Grocery shopping", "January salary").
   - **Amount**: Amount (always positive; the system marks it as expense or income based on type).
   - **Type**: Transaction type (expense, income, transfer).
   - **Category**: Category (the system may auto-suggest one).
   - **Account**: Associated account.
   - **Rate**: Exchange rate if the transaction is in another currency (default 1).
4. Click **Save**.

## Creating Records with the AI Chat

You can ask the AI chat to create transactions for you. Examples:

- "Add a $25 expense for gas today"
- "I received $1500 salary this month"
- Send a photo of a receipt and the AI will automatically extract the data.

The AI will always ask for confirmation before creating records.

## Importing Transactions from a File

1. Go to the **Import** section in the sidebar.
2. Select the file (supported formats: CSV, XLSX, XLS, ODS).
3. Map the file columns to BudgetBee fields:
   - Date, Name, Amount, Type, Category, Account.
4. Review the data and confirm the import.

## Searching and Filtering Transactions

In the records view you can:
- **Search** by name or description.
- **Filter** by date, category, account, type.
- **Sort** by date, amount, etc.

You can also ask the AI chat: "Search for transactions containing 'Amazon'".

## Editing or Deleting a Record

1. Find the record in the list.
2. Click on it to edit.
3. Modify the fields or click **Delete** to remove it.
