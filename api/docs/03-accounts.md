# Managing Accounts

## What is an Account?

An account in BudgetBee represents a source of money: bank account, cash, credit card, savings account, investment, etc.

## Account Types

BudgetBee supports the following account types:

| Type | Description |
|------|-------------|
| **Checking** | Main bank account for daily expenses |
| **Savings** | Savings account |
| **Credit Card** | Credit card |
| **Investment** | Investment or brokerage account |
| **Cash** | Physical cash |
| **Other** | Other type of account |

## Creating an Account

1. Go to the **Accounts** section in the sidebar.
2. Click the **+** button or **Add Account**.
3. Fill in the fields:
   - **Name**: Descriptive name (e.g., "Chase Checking", "Wallet cash").
   - **Type**: Account type.
   - **Currency**: Account currency.
   - **Initial Balance**: Starting balance.
4. Click **Save**.

## Editing or Deleting an Account

1. Go to the accounts list.
2. Click on the account you want to modify.
3. Edit the fields or click **Delete** to remove it.

> ⚠️ **Important**: Deleting an account will also delete ALL transactions associated with that account.

## Total Balance

The total balance is calculated by summing up the balance of all your accounts (converted to your main currency). You can view it on the main dashboard or ask the AI chat: "What's my total balance?"

## Transfers Between Accounts

To transfer money between accounts:
1. Create a new transaction of type **Transfer**.
2. Select the source account and the destination account.
3. The system will automatically adjust balances.
